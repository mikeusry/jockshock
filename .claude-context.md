# JockShock Project Context

## Project Overview
Single-product ecommerce website for JockShock - clinical-strength odor elimination spray for athletic equipment. Built with Astro + Tailwind + Shopify backend, deployed to Vercel.

**GitHub Repo**: https://github.com/mikeusry/jockshock.git
**Shopify Store**: southland-organics.myshopify.com
**Stack**: Astro 5.1, Tailwind CSS v4, Shopify Storefront API 2025-10

### Technical Notes & Gotchas
- **Shopify API Permissions**: Public Storefront token cannot access `quantityAvailable` field - requires `unauthenticated_read_product_inventory` permission
- **Background Images**: Use `background-attachment: fixed` for parallax, but test performance on mobile
- **Z-index Layering**: Hero uses `::before` pseudo-element for gradient overlay, content needs `position: relative; z-index: 1`
- **Pricing Section**: Uses negative `top: -25px` to create overlapping/lifted effect
- **Logo Filters**: `invert(1)` for color swap, `drop-shadow()` for glow effects
- **Cloudinary Public IDs**: Include the random suffix (e.g., `beckham2_t9b1zk`) not just filename
